# Promo Code

A T3 Stack app built with Next.js, tRPC, Drizzle ORM, NextAuth, and Tailwind CSS.

## Quick Start

```bash
cp .env.example .env   # fill in DATABASE_URL and AUTH_SECRET
pnpm install
pnpm db:push
pnpm dev               # http://localhost:3000
```
